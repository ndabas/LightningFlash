Here's a small description of the files included in the ZIP:

*.htc: These are the actual behaviors I have used.
Launcher.htm: Open this file to see a demo application.
Debug.htm: Provides a DHTML equivalent of TRACE. See the TRACE function in 
  CommandBars.js.
Other .htm files: These implement dialog boxes for the demo application.
*.css: Styles.
CommandBars.js: Provides global functions for all behaviors.
